# Рекомендации

Эта директория содержит сгенерированные рекомендации различных типов.

**Файлы:**
- `top_popular.parquet` - топ популярных треков
- `personal_als.parquet` - персональные рекомендации на основе ALS
- `similar.parquet` - похожие треки (item-to-item рекомендации)
- `recommendations.parquet` - итоговые ранжированные рекомендации

**Примечание:** Файлы рекомендаций хранятся в S3 и не включены в репозиторий из-за большого размера.

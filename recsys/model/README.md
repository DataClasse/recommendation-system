# Модели

Эта директория содержит обученные модели для системы рекомендаций.

**Файлы:**
- `als_model.pkl` - обученная модель ALS (Alternating Least Squares)
- `user_encoder.pkl` - энкодер идентификаторов пользователей
- `item_encoder.pkl` - энкодер идентификаторов треков
- `catboost_ranker.pkl` - обученная модель CatBoostRanker для ранжирования

**Примечание:** Файлы моделей хранятся в S3 и не включены в репозиторий из-за большого размера.

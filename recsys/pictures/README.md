# Графики EDA

Эта директория содержит графики исследовательского анализа данных (EDA).

**Файлы:**
- `user_activity.png` - распределение прослушиваний по месяцам
- `top_tracks.png` - топ-20 самых популярных треков
- `top_genres.png` - топ-20 самых популярных жанров

**Примечание:** Графики генерируются при выполнении ноутбука и не включены в репозиторий.

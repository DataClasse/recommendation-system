# Данные

Эта директория содержит подготовленные данные для построения рекомендаций.

**Файлы:**
- `items.parquet` - данные о музыкальных треках с метаданными
- `events.parquet` - данные о взаимодействиях пользователей с треками

**Примечание:** Файлы данных хранятся в S3 и не включены в репозиторий из-за большого размера.
